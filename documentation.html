Electric Vehicle Visualization Documentation

Dataset: Washington State EV Population Data

---

Design Process

The visualization was developed through an iterative process focused on improving clarity, usability, and interaction. Early versions included multiple charts, which made the interface visually complex and difficult to interpret. Users had to mentally connect different views, which increased cognitive effort.

To address this, the design was simplified into two coordinated views: a timeline on the left and a ranking chart on the right. This structure creates a clear relationship between user action and result. The timeline acts as input, while the bar chart reflects the output.

Initial sketches were used to explore layout and interaction before arriving at the final design.

---

Rationale of Design Choices

Color was used as a meaningful encoding system. Purple represents all electric vehicles, teal represents battery electric vehicles, and blue represents plug-in hybrid vehicles. These choices align with perception: teal is associated with sustainability, while blue reflects hybrid systems.

Inactive elements are shown in grey to reduce visual noise and improve focus. This creates a clear visual hierarchy and helps users quickly identify relevant data.

The interaction design uses direct manipulation. Users drag across the timeline to filter data and click on brands to explore specific trends. This removes the need for external controls and makes the interaction intuitive.

Logos were added to improve recognition. Users can identify brands instantly without reading text, which reduces cognitive effort and improves scanning speed.

A color gradient was applied to the bar chart to reinforce ranking. Darker colors represent higher values, while lighter colors represent lower values. This allows users to interpret differences visually without relying only on numbers.

---

Interaction Design

  • Drag across the timeline to select a year range
  • Bar chart updates based on selected range
  • Click a brand to highlight and view its trend
  • Hover to view detailed values

The bar chart is empty by default to guide users to interact with the timeline first.

---

Annotations and Context

Annotations were added to provide context for key events:

  • 2010: Nissan Leaf and Chevy Volt launch
  • 2017: Tesla Model 3 release
  • 2022: Inflation Reduction Act

These annotations help connect changes in the data to real-world events.

---

Insights and Findings

  • EV adoption increases significantly after 2017
  • Tesla becomes dominant in recent years
  • Earlier years show more balanced competition
  • Policy changes align with growth trends

The timeline interaction allows comparison between time periods and reveals how brand rankings change over time.

---

Implementation Notes

The visualization was built using D3.js. The dataset was preprocessed into a smaller aggregated format to improve performance. This reduced loading time and ensured smooth interaction.
